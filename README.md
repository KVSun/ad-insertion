## Requirements
- [Apache](https://httpd.apache.org/)
- [PHP](https://secure.php.net/)
- [MySQL](https://dev.mysql.com/) or [MariaDB](https://mariadb.org/)


**Update using:**

- `git pull upstream master`  
- `git submodule update --init --recursive`
- `npm run build:all`
